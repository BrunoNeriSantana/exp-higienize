# Pizzaria — Mobile-first Delivery Demo

Projeto simples de front-end focado em mobile para um site de delivery de pizzaria.

Como usar

- Abra `index.html` em um navegador moderno (Chrome, Edge, Firefox).
- O site é responsivo e mobile-first; a interface permite adicionar pizzas ao carrinho e enviar um pedido (simulado).

Próximos passos sugeridos

- Integrar gateway de pagamento (PagSeguro, MercadoPago, Stripe)
- Conectar API para receber pedidos no back-end
- Adicionar imagens e cardápio dinâmico
